# InfraRisk AI

## Project Overview

InfraRisk AI is an AI-powered Infrastructure Risk Assessment System developed to predict bridge risk levels using Machine Learning.

The project utilizes the Florida Bridge Dataset and applies an XGBoost classification model to identify bridges as Low, Medium, or High Risk.

The primary objective is to support infrastructure authorities by enabling data-driven maintenance planning and prioritizing bridges that require immediate attention.

---

## Objectives

- Predict bridge infrastructure risk.
- Improve maintenance decision making.
- Reduce infrastructure failure risk.
- Demonstrate an end-to-end Machine Learning workflow.
