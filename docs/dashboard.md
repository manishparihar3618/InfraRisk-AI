# Dashboard

The Streamlit dashboard provides an interactive interface for bridge risk assessment.

## Features

### Home

- Project Overview
- KPI Cards
- Model Comparison
- Dataset Preview

---

### Prediction

- User Input
- AI Prediction
- Confidence Score
- Risk Classification

---

### Dataset Analytics

- Risk Distribution
- Year Built Analysis
- Average Daily Traffic Analysis
- Correlation Heatmap

---

### Feature Importance

Displays the Top 15 most important features learned by the XGBoost model.

### About

Project information and technology stack.
